Для добавления плашки автора на сайт необходимо скопировать этот код:


<div data-author class="author"></div>
<script src="app.js"></script>


Вставить его в желаемое место и указать корректный путь до файла app.js. После этого плашка автора появится на странице.

В файле app.js находится html код, в котором указаны классы блока author. При необходимости их можно стилизовать через файл global.css проекта.


Позиционная стилизация может выглядеть примерно так:


.author {
	position: absolute;
	bottom: 10px;
	left: 30px;
}


Используя абсолютное позиционирование для автора можно избежать проблем с готовой вёрсткой, добавив родителю position: relativ;
